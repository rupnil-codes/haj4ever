# Haj4ever

Code. Ship. Get a blåhaj.

Haj4ever is a Hack Club YSWS (You Ship, We Ship) run by [@Flux3tor](https://github.com/Flux3tor) and [@Rupnil](https://github.com/rupnil-codes) in partnership with Hack Club. Build any project, log your hours, and earn a smolhaj, a blåhaj, or even both.

## How it works

1. **Build anything.** Make a project. It can be an app, game, website, hardware project, artwork, or anything else you create.
2. **Track your hours.** Use Hackatime to log your coding time. Complete:
   * **4 hours** = Smolhaj
   * **6 hours** = Blåhaj
   * **10 hours** = Both Smolhaj + Blåhaj
   * **More than 10 hours** = Cool secret prize
3. **Submit your project.** Send us your project and Hackatime link through the submission form. Once it's reviewed and approved, we'll ship your prize.

If shipping isn't available in your country or region, we'll work with you to find an alternative.

## Running locally

```bash
npm install
npm run dev
```
